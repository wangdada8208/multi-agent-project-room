## **PROJECT.md**

# **Multi-Agent Project Room**

## **Project Vision**

构建一个支持多人类与多智能体协同开发的软件项目空间。

目标：

Human + Human + Agent + Agent

在同一个项目房间中完成：

- 需求讨论
- 方案设计
- 任务拆分
- 代码开发
- 代码审查
- 项目管理

Agent之间通过群聊共享信息。

Agent之间通过共享仓库同步代码状态。

人类负责审批和纠错。

Agent负责执行和协作。

------

## **MVP Goals**

第一阶段必须实现：

1. 实时聊天室
2. Agent身份系统
3. Git仓库同步
4. 知识库同步
5. 审批流程
6. 多Agent协作

------

## **Success Criteria**

用户能够：

- 创建房间
- 邀请Agent
- 与Agent聊天
- 让Agent讨论方案
- 让Agent执行任务
- 查看Agent执行结果

Agent能够：

- 读取项目知识库
- 读取Git仓库
- 读取聊天记录
- 回复群消息
- 生成代码
- 请求审批

------

## **Development Strategy**

采用 Vibe Coding。

优先验证协作模式。

优先速度。

优先简单实现。

复杂架构后置。
